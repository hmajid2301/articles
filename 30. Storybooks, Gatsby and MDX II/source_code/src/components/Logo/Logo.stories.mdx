## Accent

You can adjust the accent (tags) color by passing the `accent` prop.

<Preview>
  <Story name="Accent Colour">
    <div>
      <Logo
        accent="gray-500"
        color="blue-500"
        text="Haseeb"
        hoverColor="blue-500"
      />
      <Logo
        accent="gray-500"
        color="black"
        text="Haseeb"
        hoverColor="blue-500"
      />
    </div>
  </Story>
</Preview>

## Text

You must set the main text for the logo, such as a name using the `text` prop.

<Preview>
  <Story name="Main Text">
    <div>
      <Logo text="Haseeb" />
      <Logo text="John Smith" />
      <Logo text="xyz.io" />
      <Logo text="Random" />
    </div>
  </Story>
</Preview>
